基礎知識ハミルトン形式の前提

↑ 目次に戻る
STEP 1｜記号の確認

この章では、それまで使っていたラグランジュ形式（座標 x と速度 \(\dot x\) を独立変数として扱う）から、ハミルトン形式（座標 x と運動量 p を独立変数として扱う）へ移ります。混乱しやすい記号をまず整理します。

記号意味
x（または q）一般化座標。直交座標とは限らず、角度や長さなど運動を表すのに便利な変数でよい。
\(\dot x\)（または \(\dot q\)）座標の時間微分＝一般化速度。dx/dt のこと。
px に共役な一般化運動量。定義は \(p=\partial L/\partial \dot x\)。x=位置なら \(p=m\dot x\)（普通の運動量）になるが、x=角度なら p は角運動量になるなど、x の取り方によって物理的な意味が変わる。
\(L(x,\dot x,t)\)ラグランジアン＝T−U（運動エネルギー−ポテンシャル）。x と \(\dot x\) を独立変数として扱う（ただし実際には \(\dot x=dx/dt\) という拘束がある）。
H(x,p,t)ハミルトニアン。x と p を完全に独立な変数として扱える点がラグランジアンと異なる（第3章末のまとめ参照）。

オイラー・ラグランジュ方程式（既習事項の確認）

ラグランジュ形式での運動方程式は次の形でした。以降の全ての問題の出発点になるので確認しておきます。

$$\frac{d}{dt}\left(\frac{\partial L}{\partial \dot x}\right) = \frac{\partial L}{\partial x}$$

ハミルトニアンの定義（ルジャンドル変換）

\(p=\partial L/\partial \dot x\) を使って、\(\dot x\) を p に置き換える手続きが「ルジャンドル変換」です。ハミルトニアンは次のように定義されます。

$$H(x,p,t) = p\dot x - L(x,\dot x,t)$$

ここで \(\dot x\) は p の関数として（\(p=\partial L/\partial \dot x\) を逆に解いて）x, p, t で表しておきます。この H を全微分して、オイラー・ラグランジュ方程式と p の定義を使うと、次の「正準方程式（ハミルトンの運動方程式）」が得られます。これは今後の全ての問題で当たり前のように使われる道具です。

$$\dot x = \frac{\partial H}{\partial p}, \qquad \dot p = -\frac{\partial H}{\partial x}$$
符号がつまずきやすい \(\dot p\) の式にだけマイナスが付くのを忘れがちです。「x と p は対等な変数だが、片方だけ符号が反転する」形になっているのが正準方程式の特徴だと覚えておくと、問1・問2のような確認問題でミスを防げます。

正準変換とは何か・なぜ考えるのか

正準変換とは、(x,p) から新しい変数の組 (X,P) へ移ったときに、新しい変数についても同じ形の正準方程式 \(\dot X=\partial H'/\partial P,\ \dot P=-\partial H'/\partial X\) が成り立つような変換のことです。うまく (X,P) を選べば、H′ がとても簡単な形（例えば P だけの関数、あるいは 0）になり、運動方程式が積分するだけで解けるようになります。これが正準変換を学ぶ動機です（第4章のハミルトン・ヤコビ方程式へつながります）。

母関数（generating function）の4パターン

正準変換を作る一番確実な方法が「母関数」を使う方法です。ノートでは W, W′, W″, W‴ という4種類の記号が出てきますが、これは新旧どちらの変数を独立変数に選ぶかで4通りに分類されているだけです（教科書によっては F₁〜F₄ と呼ばれます）。対応関係を先にまとめておきます。

ノートの記号独立変数関係式
\(W(x,X,t)\)古い座標 x・新しい座標 X\(p=\partial W/\partial x,\ -P=\partial W/\partial X\)
\(W'(p,X,t)\)古い運動量 p・新しい座標 X\(x=-\partial W'/\partial p,\ -P=\partial W'/\partial X\)
\(W''(x,P,t)\)古い座標 x・新しい運動量 P\(p=\partial W''/\partial x,\ X=\partial W''/\partial P\)
\(W'''(p,P,t)\)古い運動量 p・新しい運動量 P\(x=-\partial W'''/\partial p,\ X=\partial W'''/\partial P\)

どのパターンでも、新旧のハミルトニアンの関係は H′ = H + (母関数の t 偏微分) です。母関数が t を陽に含まなければ H′=H（ハミルトニアンの値は変わらない）となります。

係数比較という考え方

問5・問7・問14などで繰り返し使われる手順は次の通りです。①ある物理量（例えば L や W）を2通りの方法で全微分する。②同じ微分 dx, dP, dt などの係数を両辺で比較する。③一致する条件から関係式（\(p=\partial W/\partial x\) など）を導く。これは、ある量が「完全微分（全微分）」として書けるかどうかを調べる操作で、熱力学のマクスウェルの関係式を導くときと同じ考え方です。

正準変換の判定条件

問9・問10・問12で使われる (3.57)・(3.64) は、「ある変換が正準変換になっているかどうか」を直接チェックするための条件です。母関数を経由せずに X(x,p), P(x,p) の形が先に与えられたとき、次のポアソン括弧（1自由度の場合はヤコビアンと同じ）を計算することで正準変換かどうかを判定できます。

$$\{X,P\}_{x,p} \equiv \frac{\partial X}{\partial x}\frac{\partial P}{\partial p} - \frac{\partial X}{\partial p}\frac{\partial P}{\partial x} = 1$$

この量は、(x,p) 平面上の微小な面積要素が (X,P) 平面でどれだけ拡大・縮小されるかを表すヤコビアン \(\partial(X,P)/\partial(x,p)\) そのものです。これが 1 になることが「dW=p dx−P dX という量が完全微分になっている」（＝正準変換になっている）ための条件です。

3章 · 問1
ハミルトニアンの保存

H(x, p, t) = H(x, p) のとき、dH/dt = 0 であることを示せ。

解説

ハミルトニアン H が陽に時間を含まない場合を考える。全微分をとると

$$\frac{dH}{dt} = \frac{\partial H}{\partial x}\dot x + \frac{\partial H}{\partial p}\dot p$$

ここで正準方程式 \(\dot x = \partial H/\partial p,\ \dot p = -\partial H/\partial x\) を代入すると

$$\frac{dH}{dt} = \frac{\partial H}{\partial x}\cdot\frac{\partial H}{\partial p} + \frac{\partial H}{\partial p}\cdot\left(-\frac{\partial H}{\partial x}\right) = 0$$
∴ H が t を陽に含まなければ、H は運動の保存量（＝エネルギー）である。
なぜこれが重要か H(x,p,t) が「陽に t を含む」とは、変数 x,p の時間変化を通じてではなく、式そのものに t が直接出てくる場合（例：ばね定数が時間で変化する）を指します。この問題は「H に t が直接出てこなければ、運動の途中で H の値は変わらない＝エネルギー保存則」という、力学の基本原理をハミルトン形式で示したものです。問14ではこの前提（H が t を陽に含む）が崩れる例を扱います。

3章 · 注2〔例〕調和振動子（P.25）
L, p, H, 正準方程式を求める（4小問）

\(T=(m/2)\dot x^2\)，\(U=(1/2)kx^2\) のとき

① ラグランジアンを書いて運動方程式を導け　② x と共役な運動量 p を導け　③ ハミルトニアンを導け　④ 正準方程式を書け

解説

① ラグランジアンと運動方程式

$$L = T - U = \frac{m}{2}\dot x^2 - \frac{1}{2}kx^2$$

オイラー・ラグランジュ方程式 \(\dfrac{d}{dt}\!\left(\dfrac{\partial L}{\partial \dot x}\right)=\dfrac{\partial L}{\partial x}\) より

$$m\ddot x = -kx$$

② 共役運動量

$$p = \frac{\partial L}{\partial \dot x} = m\dot x$$

③ ハミルトニアン（ルジャンドル変換 \(H=p\dot x-L\)）

$$H = p\dot x - L = m\dot x^2 - \left(\frac{m}{2}\dot x^2 - \frac{1}{2}kx^2\right) = \frac{m}{2}\dot x^2 + \frac{1}{2}kx^2 = \frac{p^2}{2m} + \frac{1}{2}kx^2$$
\(\dot x\) を p で置き換える手順 \(H=p\dot x-L\) を計算した直後の式にはまだ \(\dot x\) が残っています（\(m\dot x^2/2+kx^2/2\) の形）。ここで②の \(p=m\dot x\) を \(\dot x=p/m\) と逆に解いて代入し、\(\dot x\) をすべて消去して p だけで書き直す、というひと手間が「ルジャンドル変換」の実体です。うっかり \(\dot x\) のまま答えにしないよう注意しましょう。

④ 正準方程式

$$\dot x = \frac{\partial H}{\partial p} = \frac{p}{m}\ \Rightarrow\ m\dot x = p \quad(イ),\qquad \dot p = -\frac{\partial H}{\partial x} = -kx \quad(ロ)$$

(イ)(ロ) より \(m\ddot x=-kx\) となり①と一致する。

注2・注3（相空間の性質） H = p²/2m + (1/2)kx² = 一定（＝E）は相空間上で楕円を表す。軌跡は等エネルギー面上を右回りに進み、互いに交わらない（初期条件が軌道を一意に決めるため）。

3章 · Q1（円筒座標, P.16）
動ポテンシャル下の質点運動（2小問）

動ポテンシャル U(z) = mgz のもとで質点が運動する。円筒座標 x = r cosφ, y = r sinφ, z = z を用いて

① ラグランジアンを書け　② r, φ, z にそれぞれ共役な運動量 \(p_r,\ p_\phi,\ p_z\) を求めよ

解説

① ラグランジアン

$$\dot x = \dot r\cos\phi - r\dot\phi\sin\phi,\qquad \dot y = \dot r\sin\phi + r\dot\phi\cos\phi$$

これをそれぞれ2乗して展開する。

$$\dot x^2 = \dot r^2\cos^2\phi - 2r\dot r\dot\phi\cos\phi\sin\phi + r^2\dot\phi^2\sin^2\phi$$ $$\dot y^2 = \dot r^2\sin^2\phi + 2r\dot r\dot\phi\sin\phi\cos\phi + r^2\dot\phi^2\cos^2\phi$$

両辺を足し合わせると、中央のクロス項（\(-2r\dot r\dot\phi\cos\phi\sin\phi\) と \(+2r\dot r\dot\phi\sin\phi\cos\phi\)）が符号違いでちょうど打ち消し合う。

$$\dot x^2+\dot y^2 = \dot r^2(\cos^2\phi+\sin^2\phi) + r^2\dot\phi^2(\sin^2\phi+\cos^2\phi)$$

ここで三角関数の恒等式 cos²φ+sin²φ=1 を使うと

$$\dot x^2+\dot y^2 = \dot r^2 + r^2\dot\phi^2$$
この展開の計算方法 \(\dot x^2+\dot y^2\) を計算するときは、上のように \(\dot x^2\), \(\dot y^2\) をそれぞれ丁寧に展開してから足すと、クロス項（\(\dot r\dot\phi\) の項）が打ち消し合い、きれいに \(\dot r^2+r^2\dot\phi^2\) にまとまります。実際に手を動かして展開しておくと、極座標に関する他の問題（問8など）でも同じ計算パターンが使えます。

運動エネルギー \(T=(m/2)(\dot x^2+\dot y^2+\dot z^2)\) にこれを代入し、ポテンシャル \(U(z)=mgz\) を引くと

$$L = T - U = \frac{m}{2}\left(\dot r^2 + r^2\dot\phi^2 + \dot z^2\right) - mgz$$

② 共役運動量

それぞれの一般化座標 r, φ, z について、共役運動量の定義 \(p=\partial L/\partial \dot q\)（座標の時間微分でLを偏微分する）を、q=r,φ,z のそれぞれについて計算する。

$$p_r = \frac{\partial L}{\partial \dot r} = \frac{\partial}{\partial \dot r}\left(\frac{m}{2}\dot r^2\right) = m\dot r$$ $$p_\phi = \frac{\partial L}{\partial \dot \phi} = \frac{\partial}{\partial \dot \phi}\left(\frac{m}{2}r^2\dot\phi^2\right) = mr^2\dot\phi$$ $$p_z = \frac{\partial L}{\partial \dot z} = \frac{\partial}{\partial \dot z}\left(\frac{m}{2}\dot z^2\right) = m\dot z$$
補足\(p_\phi=mr^2\dot\phi\) は角運動量に相当する（半径 r・速度 \(r\dot\phi\) の円運動をする質点の角運動量 \(mr\cdot(r\dot\phi)\) と同じ形）。

参考書 P.138 問9-1
L, p, H, 正準方程式（4小問）

\(L(\ell,\dot\ell)=(1/2)m\dot\ell^2-mg\ell\) のとき

① ラグランジュ方程式　② ℓ の共役運動量　③ ハミルトニアン　④ 正準方程式

解説

① ラグランジュ方程式

\(L(\ell,\dot\ell)=(1/2)m\dot\ell^2-mg\ell\) を \(\dot\ell\) で偏微分すると（ℓ を定数とみなして \(\dot\ell\) だけの関数として微分する）

$$\frac{\partial L}{\partial \dot\ell} = \frac{\partial}{\partial \dot\ell}\left(\frac{1}{2}m\dot\ell^2\right) - \frac{\partial}{\partial \dot\ell}(mg\ell) = m\dot\ell - 0 = m\dot\ell$$

次に L を ℓ で偏微分すると（今度は \(\dot\ell\) を定数とみなす）

$$\frac{\partial L}{\partial \ell} = \frac{\partial}{\partial \ell}\left(\frac{1}{2}m\dot\ell^2\right) - \frac{\partial}{\partial \ell}(mg\ell) = 0 - mg = -mg$$

オイラー・ラグランジュ方程式 \(\dfrac{d}{dt}\!\left(\dfrac{\partial L}{\partial \dot\ell}\right)=\dfrac{\partial L}{\partial \ell}\) に、左辺は \(\dfrac{d}{dt}(m\dot\ell)=m\ddot\ell\)（m は定数なので時間微分の外に出せる）、右辺は −mg を代入すると

$$m\ddot\ell = -mg$$

② 共役運動量

共役運動量の定義 \(p=\partial L/\partial \dot\ell\) は①ですでに計算済みなので、そのまま

$$p = m\dot\ell$$

③ ハミルトニアン

ハミルトニアンの定義 \(H=p\dot\ell-L\) に、まず \(p=m\dot\ell\) を \(\dot\ell=p/m\) と逆に解いてから代入していく。まず \(p\dot\ell\) の項は

$$p\dot\ell = p\cdot\frac{p}{m} = \frac{p^2}{m}$$

次に \(L=(1/2)m\dot\ell^2-mg\ell\) の中の \(\dot\ell^2\) にも \(\dot\ell=p/m\) を代入すると

$$\frac{1}{2}m\dot\ell^2 = \frac{1}{2}m\left(\frac{p}{m}\right)^2 = \frac{1}{2}\cdot\frac{p^2}{m} = \frac{p^2}{2m}$$

したがって

$$H = p\dot\ell - L = \frac{p^2}{m} - \left(\frac{p^2}{2m}-mg\ell\right) = \frac{p^2}{m} - \frac{p^2}{2m} + mg\ell = \frac{p^2}{2m} + mg\ell$$

④ 正準方程式

H=p²/2m+mgℓ を p で偏微分する（ℓ を定数とみなす）と

$$\dot\ell = \frac{\partial H}{\partial p} = \frac{\partial}{\partial p}\left(\frac{p^2}{2m}\right) = \frac{2p}{2m} = \frac{p}{m}$$

H を ℓ で偏微分する（p を定数とみなす）と

$$\dot p = -\frac{\partial H}{\partial \ell} = -\frac{\partial}{\partial \ell}(mg\ell) = -mg$$

この2式から p を消去して1本の式にする。\(\dot\ell=p/m\) の両辺を時間で微分すると \(\ddot\ell=\dot p/m\)。ここに \(\dot p=-mg\) を代入すると

$$\ddot\ell = \frac{\dot p}{m} = \frac{-mg}{m} = -g \quad\Rightarrow\quad m\ddot\ell = -mg$$
①と同じ結果 \(m\ddot\ell=-mg\)（自由落下の運動方程式）が得られた。相空間 (ℓ, p) では \(H=p^2/2m+mg\ell=\)一定 という式が、ℓ を横軸・p を縦軸にとったときの放物線を描く。

(3.42) の確認

\(W''=\displaystyle\sum_i P_iX_i + W(x,X,t) = W''(x,P,t)\)　ただし \(-P_i=\partial W/\partial X_i\) とするとき、(3.42) を確かめよ。

解説
この問題のねらい 基礎知識のところで紹介した「\(W(x,X,t)\) 型」の母関数から、「\(W''(x,P,t)\) 型」の母関数を作る手続きを確認する問題です。X をそのまま使うのではなく、\(P=-\partial W/\partial X\) を使って X の代わりに P を独立変数にする（ルジャンドル変換）というのが \(\sum_i P_iX_i\) を足す操作の意味です。

W″ を全微分すると

$$dW'' = \sum_i X_i dP_i + \sum_i p_i dx_i - (H-H')dt$$
この式の作り方 \(W=\sum_i p_iX_i\) に対して \(p\,dx-P\,dX-(H-H')dt=dW\) （母関数 W の定義そのもの）を思い出し、\(W''=\sum_i P_iX_i+W\) の両辺を全微分します。\(d(\sum_i P_iX_i)=\sum_i X_idP_i+\sum_i P_idX_i\) となるので、dW 中の \(-\sum_i P_idX_i\) の項とちょうど打ち消し合い、\(\sum_i X_idP_i\) だけが残ります。この「打ち消し合わせて欲しい変数を消す」のがルジャンドル変換の目的です。

一方、W″(x,P,t) を偏微分の形で書くと

$$dW'' = \sum_i \frac{\partial W''}{\partial P_i}dP_i + \sum_i \frac{\partial W''}{\partial x_i}dx_i + \frac{\partial W''}{\partial t}dt$$

係数を比較して

$$X_i = \frac{\partial W''}{\partial P_i},\qquad p_i = \frac{\partial W''}{\partial x_i},\qquad -(H-H') = \frac{\partial W''}{\partial t}$$

恒等変換の確認

\(W''=\displaystyle\sum_i P_iX_i\) が恒等変換になることを確かめよ。

(3.43) の導出

\(W'''=W-\displaystyle\sum_i p_ix_i+\sum_i P_iX_i = W'''(p,P,t)\) を導け。

解説
この問題のねらい 今度は x も P も独立変数から外し、p と P だけで書けるようにしたいので、\(-\sum_i p_ix_i\)（xを消す）と \(+\sum_i P_iX_i\)（Xを消す）の両方をまとめて W に足します。基礎知識でまとめた「\(W'''(p,P,t)\) 型」の母関数がここで作られます。

\(W'''=W-\sum_i p_ix_i+\sum_i P_iX_i\) を全微分すると、\(dW=p\,dx-P\,dX-(H-H')dt\) と \(d(-\sum_i p_ix_i)=-\sum_i x_idp_i-\sum_i p_idx_i\)，\(d(\sum_i P_iX_i)=\sum_i X_idP_i+\sum_i P_idX_i\) を足し合わせて、p dx の項と −P dX の項がちょうど打ち消し合い

$$dW''' = -\sum_i x_i\,dp_i + \sum_i X_i\,dP_i - (H-H')dt$$

これを W‴(p,P,t) の全微分 \(dW'''=\sum_i(\partial W'''/\partial p_i)dp_i+\sum_i(\partial W'''/\partial P_i)dP_i+(\partial W'''/\partial t)dt\) と係数比較すると

$$x_i = -\frac{\partial W'''}{\partial p_i},\qquad X_i = \frac{\partial W'''}{\partial P_i}$$
これは問9で使われている関係式 \(x=-\partial W'''/\partial p,\ X=\partial W'''/\partial P\) と一致する。x, X がともに p, P だけの関数として（独立変数の x を経由せずに）表せる点が、この母関数の意味である。
まとめ（ノートの結論）L では \(\dot x\) と x は独立ではない（拘束 \(\dot x=dx/dt\) あり）。H では x と p は完全に独立な変数として扱える。これが正準形式の本質的な利点。

例題3.1（P.39）
極座標への正準変換

(x, y) → (X, Y) = (r cosφ, r sinφ) の正準変換を、ルジャンドル型母関数 \(W'=W'(p,X,t)\)（\(p=\partial W/\partial x\) より \(W'=-px+W\)）を用いて実行し、\(T=(1/2m)(p_x^2+p_y^2)\) を極座標の運動量で表せ。

解説

母関数を \(W' = -xp_x - yp_y = -r\cos\phi\, p_x - r\sin\phi\, p_y\) とおく。\(X=-\partial W'/\partial p\) より

$$p_r = p_x\cos\phi + p_y\sin\phi,\qquad p_\phi = -r\sin\phi\, p_x + r\cos\phi\, p_y\ (=xp_y-yp_x：\text{角運動量})$$
なぜマイナス符号がつくのか 基礎知識の表にある通り、\(W'(p,X,t)\) 型では \(x=-\partial W'/\partial p\) という関係を使います。\(W'=-xp_x-yp_y\) という式自体がすでに「xをpの関数として消すための」ルジャンドル変換の形になっているため、通常のW型（マイナスなし）と符号の付き方が違って見えます。ここで焦らず基礎知識の対応表と照らし合わせるのがコツです。

\(\partial W'/\partial t=0\) なので \(H'(r,\phi,t)=H(x,y,t)\)（母関数が時間を陽に含まないのでハミルトニアンの値は変わらない）。したがって、T の式の中の \(p_x, p_y\) を \(p_r, p_\phi\) で表せばよい。まず \(p_r^2,\ p_\phi^2/r^2\) をそれぞれ計算すると

$$p_r^2 = (p_x\cos\phi+p_y\sin\phi)^2 = p_x^2\cos^2\phi + 2p_xp_y\cos\phi\sin\phi + p_y^2\sin^2\phi$$ $$\frac{p_\phi^2}{r^2} = \frac{(-r\sin\phi\,p_x+r\cos\phi\,p_y)^2}{r^2} = p_x^2\sin^2\phi - 2p_xp_y\sin\phi\cos\phi + p_y^2\cos^2\phi$$

この2式を足すと、クロス項（\(\pm 2p_xp_y\cos\phi\sin\phi\) の項）が符号違いで打ち消し合い

$$p_r^2 + \frac{p_\phi^2}{r^2} = p_x^2(\cos^2\phi+\sin^2\phi) + p_y^2(\sin^2\phi+\cos^2\phi) = p_x^2 + p_y^2$$

ゆえに \(p_x^2+p_y^2=p_r^2+p_\phi^2/r^2\) が成り立つので、これを \(T=(1/2m)(p_x^2+p_y^2)\) に代入すると

$$T = \frac{1}{2m}(p_x^2+p_y^2) = \frac{1}{2m}\left(p_r^2 + \frac{1}{r^2}p_\phi^2\right)$$
簡単な方法これは極座標運動エネルギーの標準形。x=r cosφ, y=r sinφ から直接 \(\dot x^2+\dot y^2=\dot r^2+r^2\dot\phi^2\) を計算し \(p_r=m\dot r,\ p_\phi=mr^2\dot\phi\) を代入しても同じ結果が最短で得られる（問題03の手順と同じ）。

例題3.2（P.35）
(X,P)=(p,−x) の正準性

(x, p) → (X, P) = (p, −x) が正準変換であることを確かめよ。

解説
この問題のねらい 「座標と運動量を入れ替える」という一見奇妙な変換も、実は正準変換になっているという有名な例です。x と p を対等に扱えるハミルトン形式ならではの性質を確認します。

母関数 \(W'''(p,P,t)=pP\) を用いる。この母関数は独立変数 p, P で書かれており、基礎知識の表の関係式 \(x=-\partial W'''/\partial p,\ X=\partial W'''/\partial P\) をそのまま適用できる。

$$X = \frac{\partial W'''}{\partial P} = \frac{\partial}{\partial P}(pP) = p,\qquad x = -\frac{\partial W'''}{\partial p} = -\frac{\partial}{\partial p}(pP) = -P$$

2つ目の式 x=−P を P について解くと P=−x となり、たしかに問題文の (X,P)=(p,−x) が母関数から導かれることが確認できる。

次に、この (X,P)=(p,−x) が実際に正準方程式を満たすことを直接確かめる。もとの正準方程式 \(\dot x=\partial H/\partial p,\ \dot p=-\partial H/\partial x\) を使うと

$$\dot X = \dot p = -\frac{\partial H}{\partial x} \quad\text{（もとの正準方程式より）}$$ $$\dot P = -\dot x = -\frac{\partial H}{\partial p} \quad\text{（もとの正準方程式より）}$$

一方、H は X=p, P=−x の変換で値が変わらない（H′(X,P)=H(x,p)）ので、x=−P, p=X を代入して H を X, P の関数とみなすと、連鎖律により

$$\frac{\partial H'}{\partial P} = \frac{\partial H}{\partial x}\cdot\frac{\partial x}{\partial P} = \frac{\partial H}{\partial x}\cdot(-1) = -\frac{\partial H}{\partial x},\qquad \frac{\partial H'}{\partial X} = \frac{\partial H}{\partial p}\cdot\frac{\partial p}{\partial X} = \frac{\partial H}{\partial p}\cdot 1 = \frac{\partial H}{\partial p}$$

これらを比較すると

$$\dot X = -\frac{\partial H}{\partial x} = \frac{\partial H'}{\partial P},\qquad \dot P = -\frac{\partial H}{\partial p} = -\frac{\partial H'}{\partial X}$$
(X,P)=(p,−x) は正準方程式 \(\dot X=\partial H'/\partial P,\ \dot P=-\partial H'/\partial X\) をどちらも満たすので、正準変数の組であることが確かめられた（(3.51)）。

(3.58) の確認

解説
この問題のねらい 正準方程式 \(\dot X=\partial H/\partial P\)（H は元の変数でも新しい変数でも同じ値をとる）を、連鎖律（chain rule）を使って「元の変数 x, p の偏微分」で書き直し、両辺を任意の H について比較することで、基礎知識の判定条件 {X,P}=1 が導かれることを確認する問題です。

$$\therefore\quad \frac{\partial X}{\partial x}\frac{\partial P}{\partial p} - \frac{\partial X}{\partial p}\frac{\partial P}{\partial x} = 1$$
これは基礎知識でまとめたポアソン括弧の条件 {X,P}=1 と同じであることが確認できる。

単振動 H = (1/2m)p² + (1/2)mω²x²、母関数 W(x,X) = (mω/2)x² cot X による正準変換について

次に（い）p=mωx cot X の両辺を2乗すると

$$p^2 = m^2\omega^2x^2\cot^2X$$

ここに（あ′）の x² を代入すると

$$p^2 = m^2\omega^2\cot^2X\cdot\frac{2P\sin^2X}{m\omega} = 2m\omega P\cot^2X\sin^2X$$

cot X = cos X/sin X なので cot²X sin²X = cos²X。よって

章末問題〔5〕
質量が変化する調和振動子の保存量

調和振動子 \(H=p^2/2m+(1/2)m\omega^2x^2\) の質量が \(m(t)=m_0e^{-\alpha t}\) で変化する場合の保存量を求めよ。（エネルギーは保存しない。ヒント：母関数 \(W(x,P,t)=\sqrt{m(t)}\,xP\) による変換）

解説
問1との違い 問1では「H が t を陽に含まなければ H は保存量」と学びました。この問題では m(t) が時間変化するため H は t を陽に含み、H そのものはもう保存しません。そこで正準変換によって、別の保存量 H′ を新しく作り出す、というのがこの問題のねらいです。

ヒントの母関数 \(W''(x,P,t)=\sqrt{m(t)}\,xP\) は、独立変数が x と P（基礎知識の表の「古い座標 x・新しい運動量 P」型）なので、関係式 \(p=\partial W''/\partial x,\ X=\partial W''/\partial P,\ H'=H+\partial W''/\partial t\) をそのまま使う。

① p を求める　W″ を x で偏微分する（P, t を定数とみなす）と

$$p = \frac{\partial W''}{\partial x} = \frac{\partial}{\partial x}\left(\sqrt{m(t)}\,xP\right) = \sqrt{m(t)}\,P$$

この式は後で使うために P について解いておく：\(P = p/\sqrt{m(t)}\)。

② X を求める　W″ を P で偏微分する（x, t を定数とみなす）と

$$X = \frac{\partial W''}{\partial P} = \frac{\partial}{\partial P}\left(\sqrt{m(t)}\,xP\right) = \sqrt{m(t)}\,x$$

③ \(\partial W''/\partial t\) を求める　\(W''=\sqrt{m(t)}\,xP\) の中で t に依存するのは \(\sqrt{m(t)}\) の部分だけなので、合成関数の微分（連鎖律）を使う。

$$\frac{\partial W''}{\partial t} = xP\cdot\frac{d}{dt}\sqrt{m(t)}$$

m(t)=m₀e−αt なので、まず \(\dot m(t)\) を計算すると

$$\dot m(t) = \frac{d}{dt}(m_0e^{-\alpha t}) = -\alpha m_0e^{-\alpha t} = -\alpha\,m(t)$$

次に \(d/dt\sqrt{m(t)}\) を、合成関数の微分公式 \(d/dt(f^{1/2})=(1/2)f^{-1/2}\dot f\) を使って計算する。

$$\frac{d}{dt}\sqrt{m(t)} = \frac{1}{2\sqrt{m(t)}}\dot m(t) = \frac{1}{2\sqrt{m(t)}}\cdot(-\alpha\,m(t)) = -\frac{\alpha}{2}\sqrt{m(t)}$$
この微分の計算方法 \(f(t)=m(t)^{1/2}\) の形の微分は、まず一般に \(df/dt=(1/2)m(t)^{-1/2}\dot m(t)\) と書けます（べき関数の微分 \(d/dx(x^{1/2})=(1/2)x^{-1/2}\) を m(t) に対して連鎖律で適用したもの）。ここに \(\dot m=-\alpha m\) を代入すると、\(m(t)^{-1/2}\cdot m(t)=m(t)^{1/2}=\sqrt{m(t)}\) となるので、結局 \(-(\alpha/2)\sqrt{m(t)}\) という簡単な形にまとまります。

したがって

$$\frac{\partial W''}{\partial t} = xP\cdot\left(-\frac{\alpha}{2}\sqrt{m(t)}\right) = -\frac{\alpha}{2}\left(\sqrt{m(t)}\,x\right)P$$

ここで②の結果 \(X=\sqrt{m(t)}\,x\) を使うと、\(\sqrt{m(t)}\,x\) をそのまま X に置き換えられる。

$$\frac{\partial W''}{\partial t} = -\frac{\alpha}{2}XP$$

④ H を X, P で表す　\(H=p^2/2m(t)+(1/2)m(t)\omega^2x^2\) に、①の \(p=\sqrt{m(t)}P\) と②の \(x=X/\sqrt{m(t)}\) を代入する。まず p² と x² をそれぞれ計算すると

$$p^2 = \left(\sqrt{m(t)}\,P\right)^2 = m(t)P^2,\qquad x^2 = \left(\frac{X}{\sqrt{m(t)}}\right)^2 = \frac{X^2}{m(t)}$$

これらを H に代入すると、どちらの項からも m(t) がきれいに約分されて消える。

$$\frac{p^2}{2m(t)} = \frac{m(t)P^2}{2m(t)} = \frac{1}{2}P^2,\qquad \frac{1}{2}m(t)\omega^2x^2 = \frac{1}{2}m(t)\omega^2\cdot\frac{X^2}{m(t)} = \frac{1}{2}\omega^2X^2$$

よって

$$H = \frac{1}{2}P^2 + \frac{1}{2}\omega^2X^2$$

⑤ H′ を求める　\(H'=H+\partial W''/\partial t\) の定義に③④の結果を代入すると

$$H' = \frac{1}{2}P^2+\frac{1}{2}\omega^2X^2 - \frac{\alpha}{2}XP$$
この H′ が新しい正準変数 (X,P) に対する保存量。もとの H は m(t) の変化により保存しないが、変数変換後の H′ は保存する。

微小正準変換の一般形

正準変換の母関数として、恒等変換 \(\sum_i p_iX_i\) に、小さなパラメータ ε をかけた項 \(\varepsilon S(x,p)\) を加えたものを考えます。

$$W'''(x,p) = \sum_i p_i x_i + \varepsilon S(x,p)$$

このとき、新しい変数と古い変数のずれ δx=X−x, δp=P−p は、母関数の微分を使って次のように書けることが分かります（教科書 (3.6x) 相当）。

$$\delta x_i = \varepsilon\frac{\partial S}{\partial p_i}, \qquad \delta p_i = -\varepsilon\frac{\partial S}{\partial x_i}$$

この S を「変換の生成子（generator）」と呼びます。S の形を変えるだけで、並進・回転・時間発展など、いろいろな種類の変換を統一的に表せるのがポイントです。

なぜ角運動量が回転の生成子になるのか（直感）

問15では \(S=\varepsilon(yp_x-xp_y)\) と角運動量に比例する S を選ぶと、δx, δy が実際に微小回転の式になることを確認します。逆に言えば「角運動量という保存量の正体は、空間の回転対称性から生じるものである」という、ネーターの定理の中心的な主張の具体例になっています。同様に、時間並進対称性からはエネルギー保存則、空間並進対称性からは運動量保存則が導かれる、というのがネーターの定理の一般論です。

問16・17で目指すこと

問16・17では、時間や座標を微小にずらす変換のもとで作用積分 \(I=\int L\,dt\) が不変（I′=I）となるための条件を調べ、「その条件を満たすためには、ある量 N が時間的に一定でなければならない」ということを示します。これは「対称性 ⇒ 保存量」というネーターの定理の一般的な証明の骨格をなぞったものです。計算は長いですが、ゴールは常に「N が保存量であることを示す」という1点にあることを意識すると迷いにくくなります。

§4.2 ネーターの定理 · Q2
保存量 N の導出①

\(t'=t+\varepsilon w(x,t)\)，\(x'=x+\varepsilon u(x,t)\) で不変な量 \(N=\varepsilon\!\left(\dfrac{\partial L}{\partial \dot x}u(x,t)-w(x,t)H\right)\) について、I′ = I となる条件が N が定数であることを示せ（\(I'=I+\oint N\)）。

解説（証明）
この証明の流れ（先に全体像） ①作用 I′ を新しい変数 t′, x′ で表す。②積分変数を t′ から t に変換し、L(x′,dx′/dt′) を ε の1次までテイラー展開する。③I′−I を計算すると、被積分関数が「ある量の時間についての完全微分」（＝境界の値だけで決まる形）にまとまることを示す。④I′=I が任意の積分区間で成り立つためには、その境界項の中身 N が時間によらない定数でなければならない、と結論する。この問題では \(L=L(x,\dot x)\) が t を陽に含まない（＝オートノマスな系である）ことを前提としている。
$$I' = \int_{t_0'}^{t_1'}dt'\,L\!\left(x',\frac{dx'}{dt'}\right),\qquad I = \int_{t_0}^{t_1}dt\,L\!\left(x,\frac{dx}{dt}\right)$$

Step 1：dt′ を dt で表す

t′=t+εw(x,t) の両辺を、もとの時刻 t で微分する。w は軌道上で x=x(t) を通じて t にも依存する合成関数なので、全微分（連鎖律）を使う。

$$\frac{dt'}{dt} = 1 + \varepsilon\frac{dw}{dt} = 1+\varepsilon\dot w,\qquad \text{ただし } \dot w \equiv \frac{\partial w}{\partial x}\dot x + \frac{\partial w}{\partial t}$$ $$\therefore\ dt' = (1+\varepsilon\dot w)\,dt$$

積分変数を t′ から t に置き換えると、積分区間は（t0′, t1′ に対応する）t0, t1 のまま書き直せるので

$$I' = \int_{t_0}^{t_1}dt\,(1+\varepsilon\dot w)\,L\!\left(x',\frac{dx'}{dt'}\right)$$

Step 2：dx′/dt′ を ε の1次まで求める

x′=x+εu(x,t) を t で微分すると、同じく連鎖律で

$$\frac{dx'}{dt} = \dot x + \varepsilon\dot u,\qquad \text{ただし } \dot u \equiv \frac{\partial u}{\partial x}\dot x + \frac{\partial u}{\partial t}$$

したがって dx′/dt′ は次の商で書ける。

$$\frac{dx'}{dt'} = \frac{dx'/dt}{dt'/dt} = \frac{\dot x+\varepsilon\dot u}{1+\varepsilon\dot w}$$

ε は微小量なので、分母を \(1/(1+\varepsilon\dot w) \approx 1-\varepsilon\dot w\) （ε² 以上の項を無視する近似）で置き換えて分子と掛け合わせる。

$$\frac{dx'}{dt'} \simeq (\dot x+\varepsilon\dot u)(1-\varepsilon\dot w) = \dot x - \varepsilon\dot x\dot w + \varepsilon\dot u - \varepsilon^2\dot u\dot w$$

ε² の項を無視すると

$$\frac{dx'}{dt'} \simeq \dot x + \varepsilon(\dot u-\dot x\dot w)$$

Step 3：L(x′,dx′/dt′) を ε の1次までテイラー展開する

\(x'=x+\varepsilon u\)，\(dx'/dt'=\dot x+\varepsilon(\dot u-\dot x\dot w)\) を、2変数関数 \(L(x,\dot x)\) のテイラー展開の公式 \(L(x+\Delta x,\dot x+\Delta\dot x)\simeq L(x,\dot x)+\dfrac{\partial L}{\partial x}\Delta x+\dfrac{\partial L}{\partial \dot x}\Delta\dot x\) に当てはめる（\(\Delta x=\varepsilon u,\ \Delta\dot x=\varepsilon(\dot u-\dot x\dot w)\) とみなす）。

$$L\left(x',\frac{dx'}{dt'}\right) \simeq L(x,\dot x) + \frac{\partial L}{\partial x}\cdot\varepsilon u + \frac{\partial L}{\partial \dot x}\cdot\varepsilon(\dot u-\dot x\dot w)$$

Step 4：I′ の被積分関数を展開し、I との差をとる

Step1 の \((1+\varepsilon\dot w)\) と Step3 の L の展開を掛け合わせ、ε の1次までを残す。

$$(1+\varepsilon\dot w)L\left(x',\frac{dx'}{dt'}\right) \simeq L(x,\dot x) + \varepsilon\dot wL(x,\dot x) + \varepsilon u\frac{\partial L}{\partial x} + \varepsilon(\dot u-\dot x\dot w)\frac{\partial L}{\partial \dot x}$$

右辺の第1項 \(L(x,\dot x)\) は I の被積分関数そのものなので、I′−I を計算すると、これは打ち消し合って ε の項だけが残る。

$$I'-I = \varepsilon\int_{t_0}^{t_1}dt\left[\dot wL + u\frac{\partial L}{\partial x} + \dot u\frac{\partial L}{\partial \dot x} - \dot x\dot w\frac{\partial L}{\partial \dot x}\right]$$

Step 5：\(\dot w\) の係数をまとめてハミルトニアンを作る

被積分関数のうち、\(\dot w\) を含む2つの項 \(\dot wL\) と \(-\dot x\dot w(\partial L/\partial \dot x)\) をまとめると

$$\dot wL - \dot x\dot w\frac{\partial L}{\partial \dot x} = \dot w\left(L-\dot x\frac{\partial L}{\partial \dot x}\right)$$

ここでハミルトニアンの定義 \(H=p\dot x-L=\dot x(\partial L/\partial \dot x)-L\) を思い出すと、括弧の中身は \(L-\dot x(\partial L/\partial \dot x)=-H\) とちょうど一致する。よって

$$\dot wL - \dot x\dot w\frac{\partial L}{\partial \dot x} = -\dot wH$$

したがって、被積分関数は次のようにまとまる。

$$I'-I = \varepsilon\int_{t_0}^{t_1}dt\left[u\frac{\partial L}{\partial x} + \dot u\frac{\partial L}{\partial \dot x} - \dot wH\right]$$

Step 6：残りの2項を1つの時間微分にまとめる

オイラー・ラグランジュ方程式 \(\partial L/\partial x=d/dt(\partial L/\partial \dot x)\) を使って、第1項 \(u(\partial L/\partial x)\) を書き換える。

$$u\frac{\partial L}{\partial x} = u\cdot\frac{d}{dt}\left(\frac{\partial L}{\partial \dot x}\right)$$

ここで積の微分公式 \(d/dt(fg)=\dot fg+f\dot g\) を、f=u, \(g=\partial L/\partial \dot x\) に当てはめると \(d/dt(u\cdot\partial L/\partial \dot x)=\dot u\cdot\partial L/\partial \dot x+u\cdot d/dt(\partial L/\partial \dot x)\) となるので、これを \(u(\partial L/\partial x)+\dot u(\partial L/\partial \dot x)\) と見比べると

$$u\frac{\partial L}{\partial x} + \dot u\frac{\partial L}{\partial \dot x} = \frac{d}{dt}\left(u\frac{\partial L}{\partial \dot x}\right)$$

これを Step5 の式に代入すると

$$I'-I = \varepsilon\int_{t_0}^{t_1}dt\left[\frac{d}{dt}\left(u\frac{\partial L}{\partial \dot x}\right) - \dot wH\right]$$

Step 7：\(-\dot wH\) も時間微分の形にする

\(L=L(x,\dot x)\) は t を陽に含まないと仮定しているので、H も t を陽に含まない。したがって問1の結果より dH/dt=0 が軌道上で成り立つ。この事実を使うと、積の微分公式から

$$\frac{d}{dt}(wH) = \dot wH + w\dot H = \dot wH + w\cdot 0 = \dot wH$$

すなわち \(\dot wH=d/dt(wH)\) がそのまま成り立つ（余分な項が出ない）。よって

$$I'-I = \varepsilon\int_{t_0}^{t_1}dt\,\frac{d}{dt}\left[u\frac{\partial L}{\partial \dot x} - wH\right]$$

Step 8：積分を実行して境界項の形にする

ある量の時間微分を時間で積分すると、その量の両端での値の差になる（微積分学の基本定理）。

$$I'-I = \varepsilon\left[u\frac{\partial L}{\partial \dot x} - wH\right]_{t_0}^{t_1} = \varepsilon\left(\frac{\partial L}{\partial \dot x}u - wH\right)\Big|_{t_0}^{t_1}$$
I′=I（作用が不変）となるためには、この境界項の中身 \(N=\varepsilon\!\left((\partial L/\partial \dot x)u(x,t)-w(x,t)H\right)\) が、t0 での値と t1 での値がどんな t0, t1 についても等しくなければならない。つまり N は運動の間ずっと一定値をとる保存量でなければならない。∎

ハミルトン・ヤコビ方程式の位置づけ

基礎知識STEP1で「H′ が簡単になるように正準変換を選べば、運動方程式が解きやすくなる」と述べました。第4章では、その理想を極限まで推し進め、新しいハミルトニアン H′ をまるごと 0 にしてしまう ような正準変換を探します。母関数として W″(x,α,t) 型（基礎知識の表の「古い座標 x・新しい運動量 P」型で、新しい運動量を α と書いたもの）を使うと、条件 H′=0 は次の偏微分方程式になります。これが「ハミルトン・ヤコビ（HJ）方程式」です。

$$\frac{\partial W''}{\partial t} + H\left(x,\frac{\partial W''}{\partial x},t\right) = 0$$

ここで \(p=\partial W''/\partial x\)（基礎知識の表の関係式）を使い、H の中の p をすべて \(\partial W''/\partial x\) に置き換えている点に注意してください。W″ は他の教科書では S（ハミルトンの主関数）と呼ばれることが多い量です。

H′=0 が嬉しい理由

H′=0 なら、新しい正準方程式は \(\dot X=\partial H'/\partial P=0,\ \dot P=-\partial H'/\partial X=0\) となり、新しい変数 X, P はどちらも時間に対して定数になります。つまり新しい座標系では「粒子は止まっている」ように見える変数の組が手に入るのです。あとは \(X=\partial W''/\partial \alpha\)（α は積分定数、基礎知識の表の P に相当）の関係式を x について解けば、x(t) が求まります。この「\(\beta=\partial W''/\partial \alpha\)」という関係式は、運動の初期条件（例えば初期位置）に対応する定数を表しています。

作用変数・角変数、断熱不変量とは

問20〜22で出てくる「作用変数」\(J=\oint p\,dx\) は、相空間での閉じた軌道が囲む面積にあたる量です（問12別解で見た J の正準不変性が基礎になっています）。J に対応する「角変数」ω は、H が J だけの関数になるとき \(\dot\omega=\partial H/\partial J=\)一定 という単純な形で時間発展し、1周期 T の間に ω がちょうど 2π 変化する量として定義されます。

「断熱不変量」とは、系のパラメータ（問20では箱の壁の位置）がゆっくり（系の運動の周期に比べて十分ゆっくり）変化するとき、作用変数 J がほとんど変化しない（厳密には変化速度の1次のオーダーでは変化しない）という性質のことです。これは初期の量子論（ボーア・ゾンマーフェルト量子化条件）でも重要な役割を果たした考え方です。

例題4.2内 · Q2
J の断熱不変性 O(v₀/v)

J が O(v₀/v) のオーダーで不変であることを示せ。

解説
ΔJ₁とΔJ₂とは何か（先に全体像） 壁がゆっくり動くと、\(J=4L\sqrt{2mE}\) の L も E も両方すこしずつ変化します。\(\Delta J_1\) は「壁がわずかに内側に動くことで長方形の横幅 L が減る」ことによる J の減少分、\(\Delta J_2\) は「衝突のたびに粒子の速さが増して縦幅 \(\sqrt{2mE}\) が増える」ことによる J の増加分です。この2つがちょうど同じオーダーで打ち消し合う、というのがこの問題の主張です。
$$E'=\frac{1}{2}mv^2\left(1+\frac{2v_0}{v}\right)^2 = E\left(1+4\frac{v_0}{v}+O\!\left(\left(\frac{v_0}{v}\right)^2\right)\right)$$

\(\sqrt{1+\varepsilon}\approx 1+\varepsilon/2+\cdots\) を用いて

$$\sqrt{2mE'}=\sqrt{2mE}\left(1+\frac{v_0}{v}+O\!\left(\left(\frac{v_0}{v}\right)^2\right)\right)$$

これを用いて \(\Delta J_1\)（壁との衝突で減る面積）と \(\Delta J_2\)（速度変化で増える面積）を計算すると、両者が \(O(v_0/v)\) のオーダーで打ち消し合い

$$\Delta J = \Delta J_1+\Delta J_2 = O\!\left(\left(\frac{v_0}{v}\right)^2\right)$$
壁がゆっくり動く（断熱的な）極限では、J は O(v₀/v) のオーダーでは変化しない（不変）。これが「断熱不変量としての作用変数」の具体例。
補足：作用・角変数の運動方程式\(H=H(J)\) のとき \(\dot J=-\partial H/\partial \omega=0\)（J=一定）、\(\dot\omega=\partial H/\partial J=\)一定\(\equiv \nu\)（\(\omega=\nu t+\omega_0\)）となり \(\partial H/\partial J=\nu=2\pi/T\) が成り立つ。

§4.6 正準変換の不変量 · Q1
ヤコビアン J = 1 の証明

(x,y) → (r,θ)：x=r cosθ, y=r sinθ の変換において、この変換のヤコビアン J = 1 を示せ。

解説（母関数を用いた証明）
この「ヤコビアン」は何を測っているか ここでのヤコビアンは、単なる (x,y)→(r,θ) の座標変換のヤコビアンではなく、相空間 (x,p)→(X,P) の正準変換のヤコビアンです。\(\partial(X,P)/\partial(x,p)\) と書き、微小な面積要素 dx dp が dX dP にどれだけ拡大・縮小されるかを表します。これが 1 になるということは「正準変換は相空間の面積を変えない」という、リウヴィルの定理につながる重要な性質を意味します。
$$J=\frac{\partial(X,P)}{\partial(x,p)}=\frac{\partial(X,P)}{\partial(x,P)}\Big/\frac{\partial(x,p)}{\partial(x,P)}$$
この式変形（連鎖律によるヤコビアンの分解） ヤコビアンには「途中の変数を経由して分解できる」という性質があります（ちょうど dy/dx=(dy/dz)/(dx/dz) のような関係の2変数版）。ここでは (x,P) を仲介する変数の組として選び、分母・分子をそれぞれ (x,P) を使って表される2×2行列式の比に書き換えています。

母関数 (3.42) の \(W(x,P,t)\) を用いて \(X=\partial W''/\partial P,\ p=\partial W''/\partial x\) とすると

$$\frac{\partial X}{\partial x}=\frac{\partial}{\partial P}\left(\frac{\partial W''}{\partial x}\right)=\frac{\partial p}{\partial P} \quad(\bigstar)$$
（★）が成り立つ理由 \(X=\partial W''/\partial P\) と \(p=\partial W''/\partial x\) は、どちらも同じ関数 W″ の1階偏微分です。偏微分の順序を入れ替えても結果が変わらない（\(\partial^2W''/\partial P\partial x=\partial^2W''/\partial x\partial P\)）という、2階微分の交換法則を使うと、\(\partial X/\partial x\) と \(\partial p/\partial P\) が実は同じ2階微分 \(\partial^2W''/\partial x\partial P\) を指していることが分かります。

これを用いてヤコビアンを計算すると

$$\frac{\partial(X,P)}{\partial(x,P)}=\begin{vmatrix}\dfrac{\partial X}{\partial x} & 0\\ \dfrac{\partial P}{\partial x} & 1\end{vmatrix}=\frac{\partial X}{\partial x},\qquad \frac{\partial(x,p)}{\partial(x,P)}=\begin{vmatrix}1 & \dfrac{\partial x}{\partial P}\\ 0 & \dfrac{\partial p}{\partial P}\end{vmatrix}=\frac{\partial p}{\partial P}$$

(★) より、この2つのヤコビアンは等しいので

$$J=\frac{\partial(X,P)}{\partial(x,p)}=1$$
もっと簡単な方法 正準変換であれば必ず J=1（面積保存則、リウヴィルの定理の一部）が成り立つことが一般論から保証されている。したがって、ある変換が正準変換であることを (3.64) や母関数の存在で示せれば、J=1 は毎回計算せずとも自動的に従う。
